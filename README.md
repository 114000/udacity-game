# udacity-game

## 运行

使用浏览器打开根目录下的 `index.html` 文件

## 游戏玩法

1. 你可以按键盘上的 `w/up`,`s/down`,`a/left`, `d/right` 键来操纵人物移动
2. 每次通过敌人的街道到达河对岸会得到一分，当分数积累到20时游戏结束，游戏难度会随着得分
的增加而增加
3. 每次与敌人碰撞后，会失去1分，并返回到出发点重新开始。
4. 游戏结束后你可以点击 `Restart` 按钮开始下一局游戏。
